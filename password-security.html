Password Security

Create and manage strong passwords to protect your accounts

Why Password Security Matters

Passwords are your first line of defense against unauthorized access to your accounts. A weak or reused password can compromise not just one account, but potentially all your accounts if a breach occurs. Hackers use password cracking tools, dictionary attacks, and leaked password databases to gain access.

Strong, unique passwords for each account are essential for protecting your identity, financial information, and personal data.

Key Security Practices

1

Never Share Passwords

Keep passwords confidential. Legitimate companies (banks, email providers, social media) will never ask for your password via email, phone, or chat. If someone asks, it's a scam.

✅ Do: Keep passwords private
❌ Don't: Share passwords with anyone
2

Use 12+ Characters

Longer passwords are exponentially harder to crack. Each additional character makes a password 30-50x more secure. Aim for at least 12-16 characters, or longer for critical accounts.

✅ Strong: "Tr0pic@lSunset2024!" (19 chars)
❌ Weak: "Pass123" (7 chars)
3

Mix Character Types

Combine uppercase letters (A-Z), lowercase letters (a-z), numbers (0-9), and special characters (!@#$%^&*). This dramatically increases password strength and makes brute-force attacks much harder.

✅ Do: Use ABc123!@#
❌ Don't: Use only lowercase letters
4

Unique Password per Account

Never reuse passwords. If one service is breached, hackers will try that password on other sites. Using unique passwords limits the damage if one account is compromised.

✅ Do: Different password for each account
❌ Don't: Use same password everywhere
5

Use a Password Manager

Password managers (Bitwarden, 1Password, LastPass, KeePass) securely store complex passwords. You only need to remember one master password. They also auto-fill passwords and generate new ones.

✅ Do: Use Bitwarden or 1Password
❌ Don't: Write passwords on sticky notes
6

Avoid Common Words

Don't use dictionary words, names, or sequential numbers. Hackers use "dictionary attacks" that test thousands of common words and variations. Custom, random combinations are much safer.

✅ Safe: "K8@pQx#vM2nL9$wR"
❌ Unsafe: "Sunshine123" or "Password2024"
7

Change Passwords Regularly

Update important account passwords every 3-6 months or immediately if you suspect compromise. Regular changes limit the window for hackers to use stolen credentials.

✅ Do: Change every 3-6 months
❌ Don't: Keep same password for years
8

Use Security Questions Wisely

Create unique, hard-to-guess answers for security questions. Don't use obvious information available on social media (birthplace, pet names, schools). Use false answers if possible.

✅ Do: Create non-obvious answers
❌ Don't: Answer truthfully with public info

Password Strength Indicators

Weak (❌)

Less than 8 characters, only lowercase, or common words

Examples: password, 123456, qwerty, abc123

Fair (⚠️)

8-11 characters with mixed case but no special characters

Examples: MyPassword, Welcome2024

Good (✓)

12+ characters with uppercase, lowercase, and numbers

Examples: MyP@ssw0rd2024

Strong (✅)

16+ characters with mixed case, numbers, and special characters

Examples: Tr0pic@lSunset#2024!, K8@pQx#vM2nL9$wR

Common Password Mistakes to Avoid

  • Using your name, birthdate, or personal information
  • Simple keyboard patterns (qwerty, 123456, asdfgh)
  • Dictionary words or common phrases
  • Repeating characters (aaaaaa, 111111)
  • Predictable substitutions (p@ssw0rd, passw0rd)
  • Using the same password for multiple accounts
  • Writing passwords down in insecure places
  • Using password variations that are too similar
  • Never updating passwords
  • Using obvious "recovery" passwords

How to Create Strong Passwords

Step 1: Take a memorable phrase (but don't use obvious ones)

Example: "I adopted my first cat in 2015"

Step 2: Take the first letter of each word and mix case

Example: iAmfC i2015 → IaMfCi2015

Step 3: Replace some letters with special characters

Example: IaM@fC!2015

Step 4: Ensure it's 12+ characters (add more if needed)

Example: IaM@fC!2015#Blue (15 chars - STRONG!)

If Your Password is Compromised

  • Change the password immediately on that account
  • Change it on any other accounts where you used the same password
  • Enable two-factor authentication if available
  • Check account activity for unauthorized access
  • Monitor related accounts for suspicious activity
  • Consider freezing your credit if banking information was exposed
  • Report the breach to the service provider
  • Visit haveibeenpwned.com to check if your email was in a data breach
